## Установка

```bash
yarn add @vuepress/theme-blog -D
# ИЛИ npm install @vuepress/theme-blog -D
```
## Подключение

```js
// .vuepress/config.js
module.exports = {
  theme: '@vuepress/blog',
  themeConfig: {
    // Please read documentation to see the available options.
  }
}
```
